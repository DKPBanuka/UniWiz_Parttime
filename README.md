## 🚀 Setup Guide

1. **Backend Setup**
   ```bash
   cd uniwiz-backend
   composer install
   ```
   - Configure your database connection in `config/database.php`.

2. **Frontend Setup**
   ```bash
   cd uniwiz-frontend
   yarn install
   yarn start
   ```

3. **Environment Variables**
   - Set up your `.env` files as needed for backend/frontend.
